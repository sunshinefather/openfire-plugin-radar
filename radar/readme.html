雷达插件说明

APK应用版本更新广播

参数名说明
to必须 接收人，如果接收广播的对象是多个请使用 ","分割开
version必须 版本信息

用户数据变动广播参数说明

参数名说明
username必须 新增 或者 删除等的 登录用户名
to必须 接收人，如果接收广播的对象是多个请使用 ","分割开
type必须 create新增 dismission移除

广播参数说明

参数名说明
context必须 广播内容，如果包含中文 请使用URLEncoder.encode(context, "UTF-8"));再传入
to必须 接收人，如果接收广播的对象是多个请使用 ","分割开
senduser必须 主送人，即发送消息的主人

返回结果说明

如果返回ok表示成功 ，其他表示失败

错误信息说明
IllegalArgumentException 参数错误
RequestNotAuthorised 没有访问权限.
radarDisabled 插件未启用
SharedGroupException Roster item can not be added/deleted to/from a shared group for operations with roster.
